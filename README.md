# Amazon_Clone
Amazon Clone (Frontend) | Responsive E-Commerce Website Clone
Technology Used: HTML5, CSS3, JavaScript, React.js
Project Type: Web Application
•	Built a responsive Amazon-inspired frontend with product listings, Search bar, Sign-in Form and Interactive cart UI.
•	Designed a mobile-friendly layout using modern CSS (Flexbox/Grid) for seamless cross-device compatibility.
•	Implemented dynamic product rendering and user interface components (e.g., navbar, product cards) with React.
•	Features: Static product pages, add-to-cart functionality (frontend-only), hover effects, smooth light and dark mode effects and a clean UI mimicking Amazon’s design.
